# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- **Automation Engine**: Workspace-centric architecture, repository discovery, capability detection.
- **Triggers**: Support for Manual and Cron triggers.
- **Event Streaming**: Kafka integration for Audit, Metrics, and Notifications.
- **Git & Build Plugins**: Core capabilities to manage repositories and build definitions.
- **Workflow Engine**: Core pipeline structures for executing arbitrary steps.
- **Docker Plugin**: Full container lifecycle management and logging via WebSockets.
- **Frontend Dashboard**: React/Vite dashboard shell.
- **Local-First Architecture**: Entire platform runs locally via Docker Compose without external dependencies.
